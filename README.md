# NayePankh Volunteer Hub

## About
A full-stack Volunteer Registration System built for NayePankh Foundation as part of the Full Stack Development Internship selection task.

## Features
- Public volunteer registration form with validation
- Google OAuth protected admin dashboard
- Search and filter volunteers by name, status, and area of interest
- Approve / Reject volunteer applications
- Export all volunteer data as CSV
- Fully responsive on mobile and desktop

## Tech Stack
- Frontend: React, React Router, Axios
- Backend: Node.js, Express.js
- Database: MongoDB Atlas (Mongoose)
- Auth: Google OAuth 2.0 + JWT (httpOnly cookies)
- Deployment: Railway

## Local Setup
1. Clone the repo
2. Run `npm run install-all` from root
3. Add .env file inside server/ (see .env.example)
4. Run server: `cd server && npm run dev`
5. Run client: `cd client && npm start`
